ModelArts servisini kullandığınız Region'ın project-name'ini yazmalısınız. Aşağıda bu ifadeye nasıl ulaşacağınız belirtilmiştir.

user_name | user_password | domain_name | project_name

user_name ->  IAM User Name'inizdir.
user_password -> IAM User şifrenizdir.
domain_name -> Account Name'inizdir.
project_name ->  {project-name} ile aynıdır.

Python ile Authentication Token Elde Etmek

Aşağıdaki kod örneğinde "AP-Singapore" Region'ında yer alan ModelArts servisimizi kullanabilmek için Authentication Token alıyoruz.

import requests
import json

Auth_Token_URL = "https://iam. ap-southeast-3.myhuaweicloud.com/v3/auth/tokens"

Body = {
  "auth" : { 
    "identity": { 
       "methods": [ "password" ] , 
        "password": { 
            "user": { 
                "name": "h00xxxx", 
                "password": "xxxxxxx", 
                "domain": { 
                    "name": "hwcxxxxxxx"
                } } } }, 
            "scope": { 
                "project": { 
                    "name": " ap-southeast-3 " 
                } } }
}
response = requests.request( "POST", Auth_Token_URL, json=Body)
X-Auth-Token = response.headers["X-Subject-Token" ]
